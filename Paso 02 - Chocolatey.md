# Aplicaciones en Chocolatey

## Instalar Chocolatey

~~~~ batch
https://chocolatey.org/install#individual
~~~~

## Adobe Reader

~~~~ batch
choco install adobereader -y
~~~~

## Git

~~~~ batch
choco install git -y
~~~~

## WinRar

~~~~ batch
choco install winrar -y
~~~~

## OBS Studio

~~~~ batch
choco install obs-studio -y
~~~~
